Assignment2.js 

    SetTitle 

Change the title for each puzzle to be display. 

    CreateGrid 

Creating a definite 16x16 size grid. 

    ChangePage 

Allow the user to switch between a list of puzzles. 

    CreatePuzzleList 

Create a list of buttons on the sidebar to switch between puzzles. 

    CreateColorList 

Create a list of color and assigning each color a number and button in a small grid. 

    GetImageData 

Draws the image on to canvas and returns pixel data. 

    ButtonClick 

Fills in the cell with a color if its the current selected color. 

    SwitchColor 

Allow the user to switch colors that was selected. 

    GetContrastColor 

Display the text color that can be legible on the background color 

    ConvertCoordsToIndex 

Setting the size of the grid. 

    LoadStorage 

Load the puzzle from the local storage. 

    SetStorage 

Setting the current puzzle into local storage. 

    CreateStorage 

Create the local storage for the current puzzle. 

    ModifyString 

Replace characters at index with new characters. 

    UpdateProgress 

Updates the progress of the puzzle completion. 

    ChangePalette 

3 options to change the background colors of the site
